# nettbutikk

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run Api server //port 3000
```
cd src/express-server
npm run devstart
```

### Run Web-application server //port 4000
```
node nettbutikkserver 
```
